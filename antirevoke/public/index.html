NoThx - Anti-revocation tool

NoThx is a VPN which is developed to prevent third-party apps from getting revoked or crashed. You don't need to jailbreak your IOS device to install this app. Nothx is compatible with iOS 12, iOS 11, iOS 10, and iOS 9 devices.

Trust the certificate

To open this app, All you need to do is,

❮ ❯
  • Go to Settings.
  • Tap general.
  • Find profile and device management option and tap it.
  • Find the NoThx app’s profile and trust it.

    • How NoThx Works?

    All these third-party applications are running on enterprise developer program. Every developer who signup for this program will get certificate which will allow to install apps on iOS devices. These certificates has some validity. If the validity of certificate is over, All the apps related to that particular certificate will automatically stopped (Either in the form of revoke or crashing).

    The function of these anti revoking applications is to stop Apple servers from communicating with these apps. App can only be revoked if the certificate validity is over. But if Apple unable to communicate with the app, It won’t know certificate validity. This will help us to use third party apps for longer periods of time.
